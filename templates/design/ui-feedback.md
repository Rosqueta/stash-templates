Review the following UI design and provide structured feedback.

Screen or feature: {{screen_or_feature}}
Design goal: {{design_goal}}

Cover the following areas:
1. Clarity — Is the purpose of each element immediately obvious?
2. Hierarchy — Does the visual weight guide the user to the right actions?
3. Consistency — Are spacing, typography, and colors consistent with the rest of the product?
4. Friction — What steps could be removed or simplified?
5. Suggestions — List 2-3 concrete improvements with reasoning.

---
Realiza una revisión de accesibilidad de {{pantalla_o_componente}} en {{nombre_producto}} con el objetivo de cumplir el nivel {{nivel_wcag_objetivo}} de las WCAG. Evalúa: contraste de color, jerarquía de encabezados, etiquetas de formulario, texto alternativo en imágenes, navegación por teclado, y compatibilidad con lectores de pantalla. Para cada problema detectado indica el criterio WCAG afectado, el impacto en el usuario y la corrección recomendada.
---EN---
Conduct an accessibility review of {{screen_or_component}} in {{product_name}} with the goal of meeting WCAG {{target_wcag_level}} level. Evaluate: color contrast, heading hierarchy, form labels, image alt text, keyboard navigation, and screen reader compatibility. For each issue found, indicate the affected WCAG criterion, the user impact, and the recommended fix.

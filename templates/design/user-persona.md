---EN---
Create a detailed user persona for {{product_name}}. User type: {{user_type}}. Key observed behaviors: {{key_behaviors}}. The persona should include: a fictional name and described photo, basic demographic data, main goals, frustrations and pain points, digital behaviors, and a representative quote from what this user would say. It should be realistic, not stereotyped.

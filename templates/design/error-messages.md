Redacta mensajes de error para {{nombre_producto}} relacionados con {{tipo_de_error}}. Tono de marca: {{tono_de_marca}}. Para cada mensaje incluye: título del error, descripción clara de lo que ocurrió, causa probable, y pasos concretos para resolverlo. Los mensajes no deben culpar al usuario, deben ser humanos y ofrecer siempre una salida. Propón al menos 3 variantes según la severidad del error.
---EN---
Write error messages for {{product_name}} related to {{error_type}}. Brand tone: {{brand_tone}}. For each message include: error title, clear description of what happened, probable cause, and concrete steps to resolve it. Messages should not blame the user, should be human, and always offer a way out. Propose at least 3 variants based on error severity.

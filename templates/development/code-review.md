Review the following {{language}} code as a senior engineer. Be direct and specific.

Code:
{{code}}

Cover:
1. Correctness — Are there bugs, edge cases, or logic errors?
2. Readability — Is the code clear and self-explanatory?
3. Performance — Are there any unnecessary operations or inefficiencies?
4. Security — Are there any obvious vulnerabilities?
5. Suggestions — List the top 3 changes you would make and why.

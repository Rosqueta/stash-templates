Escribe la documentación de tipo {{tipo_de_documentacion}} para {{nombre_del_componente}}, dirigida a {{audiencia}}. La documentación debe incluir: descripción general y propósito, requisitos previos, instrucciones de uso paso a paso con ejemplos de código, parámetros o props con sus tipos y valores por defecto, casos de uso comunes, y errores frecuentes con sus soluciones. Usa un lenguaje claro y ejemplos reales.
---EN---
Write {{documentation_type}} documentation for {{component_name}}, aimed at {{audience}}. The documentation should include: general description and purpose, prerequisites, step-by-step usage instructions with code examples, parameters or props with their types and default values, common use cases, and frequent errors with their solutions. Use clear language and real examples.
